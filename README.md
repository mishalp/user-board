# Getting started

### 1. Clone this repository
```
git clone https://github.com/mishalp/user-board.git
cd user-board
```

### 2. Install dependencies
```
cd frontend
npm i
cd ../backend
npm i
```

### 3. Run the server
```
npm run dev
```

### 4. Run front end
```
cd ../frontend
npm run dev
```

webpage is running at http://localhost:5173
